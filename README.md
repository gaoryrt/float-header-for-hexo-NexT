# 安装 install
```
cd /hexo/themes/next/source/css/_schemes
git clone https://github.com/gaoryrt/float_header_for_hexo_NexT.git
```
  

Rename` Mist` folder to ` Mist_without_float_haeder`  
Rename`float_header_for_hexo_NexT` to ` Mist`  
Done.
